# Professional Growth & Development Portfolio

A modern, high-fidelity, highly interactive Single Page Application (SPA) designed for Leicester High School staff to manage their professional growth, CPD, journal entries, and annual cycle checkpoints.

## Philosophy
Unlike rigid "Performance Management" software designed for compliance, this portfolio centres around **coaching, growth, and continuous reflection**. It serves as an active journal that aggregates evidence naturally throughout the academic year.

## Key Features
1. **Professional Profile**: Capture credentials, roles, and align to standard frameworks (e.g., Teachers' Standards, Leadership Standards, Support Staff framework).
2. **Goals & Priorities**: Establish unlimited goals, tag them by domain, and continuously add evidence/updates.
3. **Practice Journal**: A professional journal structured around key coaching questions (Reflection, Impact, Practice, Contribution, Growth, Support).
4. **CPD Log (Learning & Development)**: Log courses, webinars, reading, or coaching with hours, cost, and a cycle-of-learning reflection.
5. **Growth Timeline**: A chronological feed aggregating all portfolio updates.
6. **Annual Review Generator**: Compiles all goal updates, CPD logs, and reflections throughout the year into an instant, professionally structured annual review draft.

## File Structure
- `index.html`: The HTML5 document containing structural elements and the view layout.
- `styles.css`: Vanilla CSS defining design tokens (sleek dark mode, colors, typography, layout, animations).
- `app.js`: Interactive app controller containing mock data, state management, event routing, local storage synchronization, and review summary generation.

## How to Run
1. Open `index.html` directly in any web browser, OR
2. Run a local development server (e.g. `npx http-server` or `live-server` in this directory).
